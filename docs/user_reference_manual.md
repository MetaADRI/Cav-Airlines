# CAVAIR User Reference Manual

## Getting Started

1. Open the application in your web browser.
2. Use the navigation menu to access Home, Services, Booking, or Contact pages.
3. Register a new account or log in to access booking and dashboard features.

## Home Page

- View featured images and video
- See live flight schedules
- Access social media links

## Services Page

- Use the dropdown to select Domestic or International services
- View service details, images, and videos

## Booking Flights

1. Go to the Booking page.
2. Fill in your name, email, phone, and ID/Passport number.
3. Upload a scanned copy/photo of your ID or Passport.
4. Select service type (Domestic/International) and number of passengers.
5. The amount will be calculated automatically.
6. Click the Submit/Book button.
7. You can view/manage bookings in your dashboard.

## Contact Page

- View airline address and contact details
- Use the embedded Google Map for location
- Search for locations/branches
- Use the contact form for inquiries

## User Dashboard

- View all your bookings
- Update your profile (email, phone)

## Admin Dashboard (For Staff)

- Manage users, flights, schedules, and bookings

## Security & Support

- All sensitive data is protected
- For account or booking issues, use the contact form or reach out via provided support channels
